# Icon 图标

图标组件

## ant Icon API

由于图标字体本质上还是文字，可以使用 `style` 和 `className` 设置图标的大小和颜色。

[详情请点击查看官方](https://ant.design/components/icon-cn/)

```jsx
<Icon type="question" style={{ fontSize: 16, color: '#08c' }} />
```

| 参数      | 说明             | 类型      | 默认值  |
|----------|------------------|----------|--------|
| type | 图标类型 | string | - |
| spin | 是否有旋转动画 | boolean | false |
